# AI服务中转代理系统

一个功能完整的AI服务中转代理系统，支持多个AI服务提供商，包括海外的OpenAI、Anthropic、Groq、Together和国内的阿里通义、百度文心、字节豆包、MiniMax、DeepSeek。

## 功能特性

- **多提供商支持**：集成9个主流AI服务提供商
- **API密钥管理**：支持添加和管理多个API密钥
- **负载均衡**：多Key轮询，提高可用性
- **限流控制**：防止API滥用
- **费用计算**：自动计算Token使用量和费用
- **余额管理**：支持充值和查询余额
- **使用统计**：详细的使用统计和分析
- **Web管理界面**：操作简单直观
- **销售机器人**：自动回复客户咨询，提供价格表
- **交易机器人**：自动处理订单，验证付款，发卡
- **分销系统**：支持创建分销渠道，自动计算佣金
- **监控系统**：定期检查系统健康状态，发送警报

## 系统架构

- **核心系统**：FastAPI框架，支持多个AI服务提供商
- **数据库**：SQLite，存储用户、订单、分销信息
- **前端**：Web管理界面，操作简单直观
- **API**：完整的RESTful API，支持所有功能

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动服务

```bash
python app_proxy.py
```

服务将在 `http://0.0.0.0:8000` 上运行

### 访问管理界面

打开浏览器访问 `http://localhost:8000`

## 配置

### 提供商API密钥

在管理界面的"添加提供商API密钥"部分，为每个提供商添加API密钥。

### 定价策略

系统内置了以下定价策略：
- 零售价比官方低20%-30%
- 套餐：10元/万Token、50元/5万Token、200元/25万Token
- 企业客户：阶梯价、专属客服、SLA保障

### 分销管理

在管理界面的"分销管理"部分，创建分销渠道，设置佣金比例，生成推广链接。

## API文档

```bash

```

### 系统状态

```bash
GET /status
```

### 系统信息

```bash
GET /info
```

### 销售机器人

```bash

```

### 订单管理

```bash
POST /api/order

```

### 付款验证

```bash
POST /api/payment/verify
```

### 分销管理

```bash
POST /api/affiliate
GET /api/affiliate/{affiliate_id}
POST /api/referral-link
```

## 部署

### 生产环境

1. 设置环境变量：`ENVIRONMENT=production`
2. 在 `config.py` 中填写真实的上游API密钥
3. 运行 `start.bat`（Windows）或 `start.sh`（Linux）启动服务

### 监控

系统会自动监控健康状态，超过错误阈值时会发送警报。详细日志记录在 `proxy.log` 文件中。

## 安全注意事项

- 请勿将API密钥暴露在代码中或提交到版本控制系统

- 设置合理的限流措施，防止API滥用
- 监控系统异常，及时处理安全事件
